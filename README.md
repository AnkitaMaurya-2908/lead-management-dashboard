# Lead Management Dashboard (Mini CRM)

A full-stack Lead Management Dashboard (Mini CRM) built as part of a fresher hiring assignment.

This application allows users to manage leads, view analytics, and filter/search data using a clean dashboard interface.

### Authentication (Demo)
- Basic client-side login
- Email + Password + Math verification

### Leads Management
- View leads in a table
- Search by name or email
- Filter by status and stage
- Server-side pagination
- Color-coded lead rows (New / Contacted / Converted)

### Analytics
- Total leads
- Converted leads
- Conversion rate
- Leads grouped by stage
